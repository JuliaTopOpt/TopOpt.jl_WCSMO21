# TopOpt.j benchmarks

This repo constains benchmarks and demos for [TopOpt.jl](https://github.com/mohamed82008/TopOpt.jl), as supplementary materials for the abstract submitted to [WCSMO-14 (2021)](https://www.wcsmo14.org/).
